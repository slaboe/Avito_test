
Microsoft добавила в офисный пакет приложение для создания сайтов Sway. Об этом компания сообщает на сайте, посвященном этому ПО.Sway позволит создавать сайты для просмотра с мобильных и настольных устройств. При конструировании веб-страницы пользователь сможет использовать текстовую, графическую и видеоинформацию, а также добавлять интерактивные элементы. Судя по рекламному ролику, программа совмещает в себе возможности уже имеющегося ПО Microsoft — такого, как Word, Excel, OneNote и PowerPoint — для создания веб-версии мультимедийного документа. По утверждению производителя, продукт не требует от пользователя специальной подготовки для работы.Созданным в Sway интернет-проектам доступны возможности облачного хранилища Microsoft OneDrive, интеграция с социальными сетями, они есть на различных устройствах.C 1995 по 2003 год в офисный пакет Microsoft Office входило приложение FrontPage для простого создания веб-страниц, но позднее оно было исключено. Сейчас на сайте принимаются заявки на тестирование пакета Sway.
