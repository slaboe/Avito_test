
Компания Microsoft 29 июля этого года выпустит финальную версию операционной системы Windows 10 в виде бесплатного обновления для устройств на Windows 7 и Windows 8.1. Об этом корреспонденту «Ленты.ру» сообщил источник в Microsoft.Последние 14 лет новые версии Windows выходили в октябре-ноябре, но на этот раз компания решила не дожидаться осени. Платформа будет доступна пользователям в 190 странах. Россия входит в этот перечень.В Windows 10 вернется привычное меню «Пуск» и появится голосовой помощник Cortana. Управлять им можно будет с помощью голосовых или текстовых команд — помощник также обеспечит возможность возобновлять работу на разных Windows-устройствах пользователя.Система биометрической идентификации Windows Hello позволит входить в систему без пароля, при помощи биометрической аутентификации на личных мобильных устройствах. Частью новой платформы станет пакет офисных приложений Office 2016 и браузер Microsoft Edge — с его помощью можно делать на веб-страницах рукописные пометки, делиться ими с друзьями и просматривать в режиме чтения.Еще одной примечательной функцией Windows 10 стала Continuum. С ее помощью при подключении дополнительного экрана и клавиатуры смартфон, по заявлению компании, можно использовать как настольный компьютер. По идее, функция Continuum должна воплощать концепцию универсальных приложений, которые подстраиваются под размер экрана и предлагают пользователям схожие ощущения от работы на разных устройствах.Операционная система Windows 10 была представлена в сентябре 2014 года. На компьютеры с Windows 7 и Windows 8.1 новую платформу можно будет установить бесплатно. Windows 10 для других устройств появится на рынке позднее, до конца этого года. По заявлению вице-президента по операционным системам Microsoft Терри Майерсона, число устройств, на которые будет установлена Windows 10, превысит один миллиард к 2018 финансовому году.
