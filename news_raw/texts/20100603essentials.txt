
Приложения в обновленной версии пакета Windows Live Essentials получат возможность напрямую подключаться к популярным интернет-сервисам, сообщается в блоге Windows.Microsoft надеется, что это позволит пользователям использовать ОС Windows 7 в качестве интерфейса для работы с популярными фото- и видеохостингами наподобие YouTube и Flickr, соцсетями (Facebook, MySpace) и сервисами электронной почты (Hotmail, GMail или Yahoo! Mail). Программы из пакета Windows Live смогут автоматически подключаться к этим сервисам.Предполагается, что при работе с популярными интернет-сервисами непосредственно с помощью приложений, а не браузера, как это делается обычно, личные данные пользователей будут лучше защищены. Таким образом, будет снижена вероятность стать жертвой фишинговых атак или получить нежелательную рекламу.В пакет бесплатных приложений Windows Live входят сервис обмена мгновенными сообщениями Messenger, почтовый сервис Mail, приложения для обработки фото и видео Photo Gallery и Movie Maker, редактор для записей в блогах Writer и инструменты синхронизации Sync и настроек доступа Parent Safety.Этот пакет предлагается пользователям бесплатно. При этом приложения из пакета являются заменой некоторых приложений, которые ранее входили в набор программ ОС Windows (например в XP и Vista), однако не были включены в версию Windows 7.Так, Windows Live Mail является заменой приложений Outlook Express, Windows Mail, а также Windows Calendar. Загрузить пакет приложений можно на сайте Microsoft. Ожидается, что бета-тестирование обновленной версии Windows Live начнется в течение нескольких недель.В середине мая Microsoft сообщала, что планирует полностью переделать сервис электронной почты Hotmail. Публичное бета-тестирование обновленного сервиса также было запланировано на лето.
