
Корпорация Microsoft встроит в операционную систему Windows Phone файловый менеджер. Об этом со ссылкой на данные сайта User Voice сообщает издание My Nokia Blog.На сайте User Voice пользователи пишут, какие функции им хотелось бы видеть в составе тех или иных продуктов, а разработчики этих продуктов им отвечают. На странице Windows Phone запрос «Добавьте в систему файловый менеджер» является одним из самых популярных: его поддержали 77,6 тысяч человек.2 декабря статус запроса изменился на «Выполнен» (англ. Completed). Администратор сайта заявил, что «функция теперь доступна в операционной системе Windows Phone 8». Предполагается, что Microsoft включит файловый менеджер в одно из очередных обновлений для операционной системы.Не исключено, что программа может войти в состав Windows Phone 8.1 — новой версии мобильной «операционки» Microsoft, которая, по неофициальным данным, выйдет в 2014 году.В настоящее время в Windows Phone нет собственного файлового менеджера. Программы для просмотра файлов и папок для этой операционной системы, однако, существуют — их выпускают сторонние разработчики. Загрузить приложения, среди которых есть как платные, так и бесплатные, можно из интернет-магазина Microsoft.Ожидается, что помимо файлового менеджера, Microsoft встроит в новые версии Windows Phone голосовую помощницу. Ассистентка носит кодовое название Cortana. Ее особенностью будет способность к самообучению. Также сообщалось, что Cortana сможет «предугадывать» пожелания пользователя.
